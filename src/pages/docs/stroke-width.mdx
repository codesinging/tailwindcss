<!-- ## Usage -->
## 使用

<!-- Use the `stroke-{width}` utilities to set the stroke width of an SVG. -->

<!-- Useful for styling icon sets like [Heroicons](http://heroicons.com/) that are drawn entirely with strokes. -->
使用 `stroke-{width}` 功能类来设置 SVG 的线条厚度。

对于像 [Heroicons](http://heroicons.com/) 这样完全用线条(strokes)绘制的图标集来说很有用。

```html emerald
<template preview>
  <div class="text-center">
    <svg class="stroke-current stroke-1 text-emerald-600 inline-block h-24 w-24" viewBox="0 0 24 24" fill="none" stroke-linecap="round" stroke-linejoin="round">
      <circle cx="12" cy="12" r="10" />
      <line x1="14.31" y1="8" x2="20.05" y2="17.94" />
      <line x1="9.69" y1="8" x2="21.17" y2="8" />
      <line x1="7.38" y1="12" x2="13.12" y2="2.06" />
      <line x1="9.69" y1="16" x2="3.95" y2="6.06" />
      <line x1="14.31" y1="16" x2="2.83" y2="16" />
      <line x1="16.62" y1="12" x2="10.88" y2="21.94" />
    </svg>
    <svg class="stroke-current stroke-2 text-emerald-600 inline-block h-24 w-24" viewBox="0 0 24 24" fill="none" stroke-linecap="round" stroke-linejoin="round">
      <circle cx="12" cy="12" r="10" />
      <line x1="14.31" y1="8" x2="20.05" y2="17.94" />
      <line x1="9.69" y1="8" x2="21.17" y2="8" />
      <line x1="7.38" y1="12" x2="13.12" y2="2.06" />
      <line x1="9.69" y1="16" x2="3.95" y2="6.06" />
      <line x1="14.31" y1="16" x2="2.83" y2="16" />
      <line x1="16.62" y1="12" x2="10.88" y2="21.94" />
    </svg>
  </div>
</template>

<svg class="stroke-current **stroke-1** text-green-600 ..."></svg>
<svg class="stroke-current **stroke-2** text-green-600 ..."></svg>
```

<!-- ## Responsive -->
## 响应式

<!-- To control the stroke width of an SVG element at a specific breakpoint, add a `{screen}:` prefix to any existing width utility. For example, adding the class `md:stroke-2` to an element would apply the `stroke-2` utility at medium screen sizes and above. -->
要在特定的断点处控制 SVG 元素的 stroke width，可以在任何现有的宽度功能类中添加 `{screen}:` 前缀。例如，添加类 `md:stroke-2` 到一个元素中，就会在中等屏幕尺寸及以上时应用 `stroke-2` 功能类。

```html
<svg class="stroke-1 **md:stroke-2** ...">
  <!-- ... -->
</svg>
```

<!-- For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation. -->
关于 Tailwind 的响应式设计功能的更多信息，请查看[响应式设计](/docs/responsive-design)文档。

<!-- ## Customizing -->
## 自定义

<!-- Control which stroke-width utilities Tailwind generates by customizing the `theme.strokeWidth` section in your `tailwind.config.js` file: -->
通过在您的 `tailwind.config.js` 文件中自定义 `theme.strokeWidth` 部分来控制 Tailwind 产生的线条宽度。

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        strokeWidth: {
+         '3': '3',
+         '4': '4',
        }
      }
    }
  }
```
